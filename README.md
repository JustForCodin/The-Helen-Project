# The Helen Project

This is Helen - my cross-platform voice assistan. It's in very early development stage and can be unstable.
There is only English language support. I started my work only week ago. There is no installer
like .deb or .exe or something else. For now it can do very specific, hardly coded stuff.
It recommended to run only in linux. But development doesn't stop and it'll have more features in the future :)


Это Хелен - мой кроссплатформенный голосовой помощник. Она на очень ранней стадии разработки и может быть нестабильной.
Есть только поддержка английского языка. Я начал свою работу только неделю назад. Там нет установщика как .deb или .exe или что-то еще. На данный момент он может делать очень специфичные,"захардкоженные" вещи.
Рекомендуется запускать только в Linux. Но разработка не останавливается, и в будущем у нее будет больше возможностей :)


Це Хелен - мій крос-платформний голосовий асистент. Вона на дуже ранній стадії і може бути нестабільною.
Є лише підтримка англійської мови. Я почав свою роботу лише тиждень тому. Немає інсталятора на зразок .deb чи .exe чи чогось іншого. Наразі він може робити дуже специфічні, важко кодовані речі.
Рекомендується запускати лише в Linux. Але розробка не припиняється, і в майбутньому вона матиме більше можливостей :)

To launch it, just clone this repository to your computer and run:

    python3 helen.py.
    
Warning! For correct work you need to install required models:
  
    pip install gtts, speech_recognition, matplotlib, numpy, scikit-learn

and also:
    
    sudo apt-get install python3-pyaudio
    
Good luck! :)
